  
 # Inline vs Block
Hay 2 tipos de eolementos HTLM: Inline, block
Flow de elementos en forma horizontal.

Ejemplos de etiquetas inline: 

- las imagenes `<img>`
- botones `<button>`
- enlaces `<a>`
- cajas `<span>`
- textos `<strong> <em> <i> <mark>`

Ejemplos de etiquetas en Bloque: 

- los div `<div>`
- parrafos p
- headers h1, h2... h6
- tablas `<table>`
- regletas `<hr>`
- listas `<ol> <ul> <li>`
